# ⚙️ K-Nearest Neighbors (KNN) Classifier From Scratch in Python

This project focuses on implementing the K-Nearest Neighbors (KNN) Classification algorithm from scratch using Python and NumPy without relying on Scikit-Learn's built-in KNN implementation.

Before building the classifier, Euclidean Distance and Manhattan Distance calculations were implemented from scratch to understand how KNN measures similarity between data points.

---

# 📌 Project Overview

K-Nearest Neighbors (KNN) is one of the simplest and most intuitive Machine Learning algorithms.

Unlike many algorithms, KNN does not learn an explicit model. Instead, it classifies new data points by finding the nearest neighboring samples in the training dataset.

This project was divided into two parts:

### Part 1

* Euclidean Distance Calculation
* Manhattan Distance Calculation
* Similarity Measurement

### Part 2

* Complete KNN Classifier Implementation
* Neighbor Selection
* Majority Voting
* Classification
