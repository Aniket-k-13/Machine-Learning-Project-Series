# ⚙️ Lasso Regression From Scratch in Python

This project focuses on implementing the Lasso Regression algorithm from scratch using Python and NumPy without relying on Scikit-Learn's built-in Lasso model for training.

The objective was to understand the mathematical foundations of L1 Regularization, Gradient Descent optimization, and how regression models learn from data.

---

# 📌 Project Overview

Lasso Regression (Least Absolute Shrinkage and Selection Operator) is a linear regression technique that introduces L1 regularization to reduce overfitting and perform feature selection.

In this project, the complete Lasso Regression algorithm was implemented manually, including:

* Weight Initialization
* Gradient Descent Optimization
* Weight Updates
* L1 Regularization Penalty
* Prediction Function

The custom implementation was later compared against Scikit-Learn's implementation.
