# ⚙️ Support Vector Machine (SVM) Classifier From Scratch

This project focuses on implementing the Support Vector Machine (SVM) algorithm completely from scratch using Python and NumPy without relying on Scikit-Learn's built-in SVM implementation.

The goal was to understand the mathematical foundations behind Support Vector Machines and how optimization-based classification algorithms work internally.

---

# 📌 Project Overview

Most Machine Learning projects use pre-built libraries that hide the internal workings of algorithms.

In this project, I implemented a Support Vector Machine Classifier from scratch to gain a deeper understanding of:

* Hyperplanes
* Margin Maximization
* Weight Optimization
* Gradient Updates
* Binary Classification

The model was trained and tested on the Diabetes Prediction dataset.
