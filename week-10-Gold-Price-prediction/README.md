# 💰 Gold Price Prediction using Machine Learning

This project focuses on predicting **gold prices** using Machine Learning techniques and historical financial market data.

---

## 🚀 Overview

Gold price prediction is an important real-world regression problem where historical financial and economic data are analyzed to predict future gold prices.

This project demonstrates a complete Machine Learning workflow including:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature analysis
- Model training
- Performance evaluation

---

## 📂 Dataset Source

Dataset used in this project:

🔗 Kaggle Dataset – Gold Price Data  
https://www.kaggle.com/datasets/altruistdelhite04/gold-price-data

---

## 🧠 Features & Concepts Used

- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Correlation analysis using heatmaps  
- Feature selection and analysis  
- Train-Test Split  
- Regression modeling  
- Model evaluation and prediction analysis  

---

## ⚙️ Workflow

1. Load dataset  
2. Perform data cleaning and preprocessing  
3. Analyze feature relationships using EDA  
4. Visualize data trends and correlations  
5. Split dataset into training and testing sets  
6. Train regression model  
7. Evaluate model performance
