# 🔥 Calories Burnt Prediction using Machine Learning

This project focuses on predicting the number of calories burned during physical activities using Machine Learning regression techniques.

The model learns relationships between various physiological and activity-related parameters such as age, weight, duration, heart rate, and body temperature to estimate calories burned.

---

# 📌 Project Overview

Accurately estimating calorie expenditure is useful for fitness tracking, health monitoring, and personalized workout planning.

In this project, a Machine Learning regression model was trained on exercise and calorie data to predict calories burned based on user and activity characteristics.

---

# 📊 Dataset Information

Dataset Source:

https://www.kaggle.com/datasets/fmendes/fmendesdat263xdemos?select=exercise.csv

The dataset contains information such as:

* Gender
* Age
* Height
* Weight
* Exercise Duration
* Heart Rate
* Body Temperature
* Calories Burned (Target Variable)

### Target Variable

* Calories Burned
