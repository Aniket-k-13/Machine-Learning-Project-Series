# 🏥 Medical Insurance Cost Prediction using Machine Learning

This project focuses on predicting medical insurance costs using Machine Learning techniques and healthcare-related data.

---

## 🚀 Overview

Medical insurance cost prediction is an important regression problem where factors such as age, BMI, smoking habits, and other medical attributes are analyzed to estimate insurance charges.

This project demonstrates a complete Machine Learning workflow including:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature encoding
- Model training
- Prediction and evaluation

---

## 🧠 Features & Concepts Used

- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Label Encoding  
- Feature Analysis  
- Train-Test Split  
- Regression Modeling  
- Prediction & Evaluation  

---

## ⚙️ Workflow

1. Load dataset  
2. Perform data preprocessing and encoding  
3. Analyze relationships between features  
4. Visualize trends and patterns  
5. Split dataset into training and testing sets  
6. Train regression model  
7. Evaluate model performance
