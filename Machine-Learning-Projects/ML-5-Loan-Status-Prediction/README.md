# 💰 Loan Status Prediction using Machine Learning

This project focuses on predicting whether a loan application will be **approved or rejected** using Machine Learning techniques.


---

## 🚀 Overview

Loan approval prediction is an important real-world application in the financial domain.  
This project uses applicant data such as income, credit history, and demographics to build a classification model.

---

## 🧠 Features

- Data preprocessing and cleaning  
- Handling missing values  
- Encoding categorical variables  
- Feature selection  
- Classification using **Support Vector Machine (SVM)**  
- Model evaluation using accuracy score  

---

## ⚙️ Workflow

1. Load dataset  
2. Perform data cleaning and handle missing values  
3. Encode categorical features  
4. Split dataset into training and testing sets  
5. Train model using SVM  
6. Evaluate model performance  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Scikit-learn  

---

## 📊 Model Details

- Algorithm: Support Vector Machine (SVM)  
- Problem Type: Binary Classification  
- Evaluation Metric: Accuracy Score  

## 📌 Key Learnings

- Handling structured real-world datasets  
- Importance of data preprocessing in ML pipelines  
- Working with categorical variables  
- Applying SVM for classification tasks
