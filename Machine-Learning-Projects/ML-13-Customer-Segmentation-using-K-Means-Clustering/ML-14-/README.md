# 🛍️ Customer Segmentation using K-Means Clustering

This project focuses on customer segmentation using the K-Means Clustering algorithm, an unsupervised Machine Learning technique used to identify distinct customer groups based on purchasing behavior.

---

## 📌 Project Overview

Customer segmentation helps businesses understand different customer groups and create targeted marketing strategies.

In this project, K-Means Clustering was used to segment customers based on features such as annual income and spending score, allowing meaningful customer groups to be identified without using labeled data.

---

## 📊 Dataset Information

Dataset Source:

https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

The dataset contains customer information including:

* Customer ID
* Gender
* Age
* Annual Income (k$)
* Spending Score (1–100)

## 🔄 Workflow

1. Load Dataset
2. Perform Data Exploration
3. Visualize Customer Features
4. Determine Optimal Number of Clusters using the Elbow Method
5. Train K-Means Clustering Model
6. Assign Customers to Clusters
7. Visualize Customer Segments

---

## 📈 Model Highlights

### K-Means Clustering

The K-Means algorithm was used to group customers into clusters based on purchasing behavior.

The Elbow Method was applied to determine the optimal number of clusters before training the final model.
