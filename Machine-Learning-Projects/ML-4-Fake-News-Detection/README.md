# 📰 Fake News Detection using Machine Learning & NLP

This project focuses on building a Machine Learning model to classify news articles as **Real or Fake** using Natural Language Processing (NLP) techniques.

---

## 🚀 Overview

Fake news detection is an important real-world application of Machine Learning and NLP.  
In this project, we process textual data and train a classification model to identify whether a news article is genuine or misleading.

---

## 🧠 Features

- Text preprocessing and cleaning  
- Stopword removal  
- Stemming using NLTK  
- Feature extraction using **TF-IDF Vectorization**  
- Binary classification using **Logistic Regression**  
- Model evaluation using accuracy score  

---

## ⚙️ Workflow

1. Load dataset  
2. Clean and preprocess text data  
3. Apply tokenization, stopword removal, and **stemming**  
4. Convert text into numerical form using **TF-IDF Vectorizer**  
5. Split dataset into training and testing sets  
6. Train model using Logistic Regression  
7. Evaluate model performance  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- NLTK  

---

## 📊 Model Details

- Algorithm: Logistic Regression  
- Feature Extraction: TF-IDF  
- Text Processing: Tokenization, Stopword Removal, Stemming  
- Evaluation Metric: Accuracy Score
