# 🧩 Autism Prediction using Machine Learning

This project focuses on predicting Autism Spectrum Disorder (ASD) using machine learning techniques and behavioral screening data.

Early detection of ASD can help provide timely support and intervention. This project uses machine learning models to analyze behavioral screening responses and demographic information to predict the likelihood of Autism Spectrum Disorder.

---

## 📌 Project Overview

The objective of this project is to develop a machine learning model capable of assisting in ASD prediction using screening questionnaire responses and related features.

The project includes:

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Encoding
- Model Training
- Model Comparison
- Hyperparameter Tuning
- Cross Validation
- Model Evaluation

---

## 📊 Dataset Information

The dataset contains behavioral screening responses and demographic information used to identify Autism Spectrum Disorder patterns.

### Features Include

- A1_Score to A10_Score
- Age
- Gender
- Ethnicity
- Jaundice History
- Family History of Autism
- Country of Residence
- Previous App Usage
- Screening Result Score
- Relationship of Respondent

### Target Variable

- ASD Prediction
  - 0 = No Autism
  - 1 = Autism Detected
