# ❤️ Heart Disease Prediction using Machine Learning

This project focuses on predicting the likelihood of heart disease using Machine Learning techniques and healthcare-related datasets.

---

## 🚀 Overview

Heart disease prediction is an important real-world healthcare application where patient medical information is analyzed to predict the presence of heart disease.

This project demonstrates a complete Machine Learning workflow including:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature analysis
- Model training
- Performance evaluation

---

## 🧠 Features & Concepts Used

- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Correlation analysis and feature study  
- Train-Test Split  
- Classification modeling  
- Model evaluation using accuracy score  

---

## ⚙️ Workflow

1. Load dataset  
2. Perform data cleaning and preprocessing  
3. Analyze feature relationships using EDA  
4. Visualize dataset patterns and correlations  
5. Split dataset into training and testing sets  
6. Train classification model  
7. Evaluate model performance  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  
- Seaborn  

---

## 📊 Model Details

- Problem Type: Classification  
- Goal: Predict Heart Disease
