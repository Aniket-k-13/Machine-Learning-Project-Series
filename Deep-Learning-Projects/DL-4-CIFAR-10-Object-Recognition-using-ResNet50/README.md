# 🖼️ CIFAR-10 Object Recognition using ResNet50

This project focuses on multi-class image classification using the CIFAR-10 dataset and Transfer Learning with ResNet50.

ResNet50 is a powerful deep convolutional neural network pre-trained on the ImageNet dataset. By leveraging Transfer Learning, the model can efficiently recognize different object categories while reducing training time and computational cost.

---

## 📌 Project Overview

The objective of this project is to classify images into one of the ten CIFAR-10 object categories using a pre-trained ResNet50 model.

The project includes:

- Data Collection
- Data Preprocessing
- Image Resizing
- Image Normalization
- Transfer Learning
- Feature Extraction
- Model Training
- Model Evaluation

---

## 📂 Dataset Information

### CIFAR-10 Dataset

Dataset Statistics:

- 60,000 Images
- 10 Classes
- 50,000 Training Images
- 10,000 Testing Images

### Classes

- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck
