# 🐶🐱 Dog vs Cat Classification using Transfer Learning

This project focuses on classifying images of dogs and cats using Deep Learning and Transfer Learning techniques.

Transfer Learning enables us to use knowledge learned from large pre-trained models and apply it to new image classification tasks, significantly reducing training time and computational requirements.

---

## 📌 Project Overview

The goal of this project is to build an image classification model capable of distinguishing between dogs and cats using a pre-trained deep learning architecture.

The project includes:

- Data Collection
- Image Preprocessing
- Image Normalization
- Transfer Learning
- Model Training
- Model Evaluation
- Image Prediction

---
