# 🚗 Car Price Prediction using Machine Learning

This project focuses on predicting the **selling price of used cars** using Machine Learning techniques based on various vehicle-related features.

---

## 🚀 Overview

Car price prediction is an important real-world regression problem where vehicle attributes such as manufacturing year, fuel type, transmission type, and kilometers driven are used to estimate the selling price of a car.

This project demonstrates a complete Machine Learning workflow including data preprocessing, exploratory data analysis, model training, and evaluation.

---

## 📂 Dataset Source

Dataset used in this project:

🔗 Kaggle Dataset – Vehicle Dataset from CarDekho  
https://www.kaggle.com/datasets/nehalbirla/vehicle-dataset-from-cardekho

## 🧠 Features Used

- Car Name  
- Year  
- Selling Price  
- Present Price  
- Kilometers Driven  
- Fuel Type  
- Seller Type  
- Transmission  
- Owner

## 🔧 Concepts & Techniques Used

- Data preprocessing and cleaning  
- Handling categorical variables using encoding  
- Exploratory Data Analysis (EDA)  
- Feature selection and transformation  
- Train-Test Split  
- Regression modeling  
- Model evaluation  

---

## ⚙️ Workflow

1. Load dataset  
2. Perform data cleaning and preprocessing  
3. Analyze feature relationships using EDA  
4. Encode categorical variables  
5. Split dataset into training and testing sets  
6. Train regression model  
7. Evaluate model performance
